Getting Started with Pagelove

January 15, 2024 by Alex Developer

Pagelove makes it easy to serve and manage your web content with advanced features like CSS Selector Range units and automatic microdata to JSON-LD conversion.

Key Features

  • Selector Range Units: Request specific parts of your HTML using CSS selectors
  • Content Negotiation: Automatically serve JSON-LD when requested
  • RESTful Operations: Update content using PUT, POST, and DELETE with selectors

Quick Examples

# Get just the hero section
curl -H "Range: selector=#hero" http://localhost:5683/

# Get this article as JSON-LD
curl -H "Accept: application/ld+json" http://localhost:5683/

# Update the tagline
curl -X PUT -H "Range: selector=.tagline" \
  -H "Content-Type: text/html" \
  -d '<p class="tagline">New tagline here</p>' \
  http://localhost:5683/
