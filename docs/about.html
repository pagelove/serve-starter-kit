Pagelove Starter Kit

This starter kit demonstrates the key features of the Pagelove platform, including CSS Selector Range units and microdata to JSON-LD conversion.

What's Included

  • Pre-configured Setup: Ready-to-use pagelove.json configuration with sensible defaults
  • Example HTML: Sample pages with semantic markup and microdata
  • Responsive Styles: Modern CSS with mobile-first design
  • Best Practices: Structured data, accessibility, and SEO considerations

Getting Started

To use this starter kit as a template for your own project: 

# Initialize a new project from this template
pagelove init -n my-app -t https://github.com/pagelove/serve-starter-kit

# Navigate to your project
cd my-app

# Start the development server
pagelove serve

Testing Features

Once your server is running, try these commands to test the features:

# Get just this article
curl -H "Range: selector=article" http://localhost:5683/about.html

# Get the navigation menu
curl -H "Range: selector=#main-nav" http://localhost:5683/about.html

# Check what methods are available
curl -X OPTIONS -H "Range: selector=#content" http://localhost:5683/about.html
